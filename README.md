# Gimp-Plugins

## What is this repository?

This repository functions as a small collection of gimp plugins that I wrote. The plugins are free to be used, modified and distributed under the GNU General Public License 3.0.

## Installation

To install any plugin, simply download its folder and place it in the appropriate GIMP plug-ins folder.
To find which folders GIMP searches for plugins, go into GIMP > Edit > Preferences > Folders > Plug-Ins.
Note that placing the plug-in files into the specified folder will not work; the file must be in a subfolder of the same name with the extension omitted.

Example Path (linux):
`~/.config/GIMP/3.2/plug-ins/my_plugin/my_plugin.py`

Example Path (Windows):
`C:\Users\User\AppData\Roaming\GIMP\3.0\plug-ins\my_plugin\my_plugin.py`
